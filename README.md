# Game-Edukasi-Ular-Tangga
Game edukasi ular tangga dirancang untuk meningkatkan keaktifan siswa dalam belajar melalui soal, tantangan (challenge), dan inovasi interaktif berbasis gestur tangan. Pemain dapat menggerakkan objek dalam permainan tanpa sentuhan fisik, sehingga proses belajar menjadi lebih menarik, interaktif, kompetitif, dan tidak membosankan.
